bridge-Bridgemate
=================
This project is developed based on Python and try to communicate with Bridgemate2.




Install on Windows
===================
- python27
- win32com python module
- Access 2010
- Bridgemate Controller Software
- BM2ServerDriver



How to use ?
============
python main.py [create/open] <PROJECT_NAME>
